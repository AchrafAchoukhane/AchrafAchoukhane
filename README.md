## 🚀 Main Project

### 🛡️ AI-SOC Banking Cybersecurity Platform

I am currently working on an intelligent cybersecurity platform designed for **early cyber threat detection, alert analysis, automated response, and post-incident reporting** in a banking cybersecurity context.

The platform combines:

- **Wazuh** as the SIEM solution
- **AI-assisted analysis** to enrich and explain security alerts
- **SOAR concepts** for graduated response actions
- **FastAPI backend** for API services and alert processing
- **Web dashboard** for security monitoring and visualization
- **Linux-based virtual lab** for realistic testing and deployment

> The main repository is private due to confidentiality, but the project is part of my cybersecurity engineering portfolio.
